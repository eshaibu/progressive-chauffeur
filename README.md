# Ruachteq Chauffeur

A [Next.js](https://nextjs.org) application for fleet management and chauffeur services.

## Getting Started

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

### Build

Build the application for production:

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Deployment with Cloudflare Pages

Deploy to Cloudflare Pages using Wrangler:

```bash
npx wrangler pages deploy ./out --project-name=[project-name]
```

Replace `[project-name]` with your actual Cloudflare Pages project name.
